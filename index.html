SaralAI — Accessibility Assistant

An AI-powered reading assistant for ADHD and Dyslexia. Built during the Accenture Hackathon, it simplifies complex web content into easily digestible formats.

Key Technical Features:

  • Dual Delivery Model: Built as both a Chrome Extension and a Progressive Web App (PWA).
  • AI Integration: Connects to Google Gemini API utilizing a robust multi-model fallback chain for high uptime processing.
  • Serverless Backend: Cloudflare Workers deployed to handle CORS routing and AI resilience logic.
  • Isolation: Uses Shadow DOM to prevent any CSS style conflicts with host pages.
  • Accessibility: Includes focus trapping, text-to-speech capabilities, and real-time translation across 12+ Indian languages.

Impact:

  • Successfully addresses accessibility gaps for visually impaired and neurodivergent users.
  • Demonstrates complete multi-stack engineering delivered efficiently within a tight hackathon timeline.
